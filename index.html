Cryptotux

Cryptotux is a Linux image preconfigured with tools for crypto assets, blockchains, and Distributed Ledger Technologies (DLT). It is designed as an educational tool and a development environment.

Cryptotux is provided as .ova images, which can be directly imported into VirtualBox (6.X+) and VMware. This enables a quick start for workshops, immediate coding, or testing new nodes in an isolated environment. It can also be installed on a server or on Windows Subsystem for Linux.
